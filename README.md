## Quadcopter Documentation

Provide documentation, instructions, and details for our Quadcopter project.

### Installation

Clone repo:
'$ git clone x'

Install gem:
`$ gem install sinatra`

Install change reloading:
`$ gem install shotgun`

Run app:
'$ shotgun -p 4567 app.rb'

Load in browser:
`http://localhost:4567/`
