## Quadcopter Documentation

Provide documentation, instructions, and details for our Quadcopter project.

### Installation

Clone repo:
'$ git clone x'

Install Sinatra.rb:
`$ gem install sinatra`

Install gems
`$ gem install shotgun`
`$ gem install httparty`
`$ gem install shotgun`

Run app:
`$ shotgun -p 4567 app.rb`

Load in browser:
`http://localhost:4567/`
